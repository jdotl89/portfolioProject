Hang Man game in JAVA

Hang Man game was a project from my first Java class. Using a culmination of all we have learned in the class such as creating variables, methods, arrays, and making the program object oriented versus having it all run on one page of code. The program would read a file consisting a variety of words, then would use a random generator to select a word and place that word into a newly created array with the length based off the length of the word. Next the user would be asked to input a letter and the program would scan for letter in the array. If correct the program would display the letter and a counter for how many guesses would increment as well, if incorrect a underscore would be displayed and a counter for a strike would increment. Reaching a number of strikes would call the game to be over or if the all letters were guessed correctly the program would congratulate the user and the game would be over as well. Over all this program utilized the practice of calling methods within a method as Java is object oriented based
